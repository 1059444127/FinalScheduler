# FinalScheduler


#### JFinal2.0 + Bootstrap + SeaJS

终极排班管理系统，改变传统的排班模式！

## 系统功能模块

#### 人员管理

对人员信息的维护，增删改查等

![人员管理][1]

#### 分组管理

对人员进行分组，对不同的任务或者部门人员分组来管理排班，分组支持增删改查等

![分组管理][2]

#### 班次设置

排班班次设置，支持自定义班次名称、颜色和时间等

![班次设置][3]

#### 排班管理

根据分组的人员和班次的设置排班，班次的颜色和班次设置同步

![排班管理][4]

#### 考勤统计

统计人员的考勤情况，正常，迟到，旷工，请假等各种可以定制的。
![考勤统计][5]
